# Weather App

A responsive weather web application built using HTML, CSS, and JavaScript that allows users to search for any city and view real-time weather information. The application integrates the OpenWeather API to fetch live weather data and displays temperature, humidity, wind speed, and weather conditions with dynamic weather icons.

---

## Features

- Search weather information by city name.
- Display real-time temperature, humidity, and wind speed.
- Show dynamic weather icons based on current weather conditions.
- Fetch live weather data using the OpenWeather API.
- Responsive design for desktop and mobile devices.
- Handle invalid city names with appropriate error messages.

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- OpenWeather API

## Installation

### Clone the repository

```bash
git clone https://github.com/your-username/Weather-App.git
```

### Navigate to the project directory

```bash
cd Weather-App
```

### Run the application

Open the `index.html` file in your preferred web browser.

---

## API Configuration

This project uses the OpenWeather API to retrieve real-time weather information.

1. Create an account on OpenWeather.
2. Generate your API key.
3. Open the `script.js` file.
4. Replace the placeholder API key with your own.

```javascript
const apiKey = "YOUR_OPENWEATHER_API_KEY";
```
